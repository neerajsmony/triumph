// Introduction to Bootstrap // ================================================================================= // ✅ What is Bootstrap? // ================================================================================= // - Bootstrap is a free and open-source front-end framework // used for designing responsive and mobile-first websites. // - Originally developed by Twitter, now maintained // by the open-source community. // - It provides pre-designed CSS and JavaScript components // for typography, forms, buttons, navigation, modals, and more. // ✅ Benefits of Using Bootstrap // ================================================================================= // - Responsive Design: Adapts content to different screen sizes // (desktop, tablet, mobile). // - Predefined Styles and Components: Speeds up development // with buttons, navbars, modals, etc. // - Grid System: Easy-to-use 12-column responsive layout system. // - Cross-Browser Compatibility: Works across all major modern browsers. // - Customizable: You can override Bootstrap styles or use Sass variables. // ✅ How Bootstrap Works // ================================================================================= // CDN (Content Delivery Network): Link directly to Bootstrap's hosted CSS/JS files. // Local Setup: Download Bootstrap files and link them locally in your project. // // // // Examples // ================================================================================= // ✅ Link Bootstrap via CDN // Create a new HTML file (e.g., index.html) // Add Bootstrap CSS and JS using CDN // Create a simple layout // // // // // Bootstrap Demo // // // //
